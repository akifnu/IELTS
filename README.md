# IELTS

This is an IELTS app.

## Shine — Lifestyle Flashcards (Web)

The main web app is in [`flashcards/web/`](flashcards/web/). Open it locally:

```bash
cd flashcards/web && ./serve.sh
```

Then visit **http://localhost:8080**

**Live site (after GitHub Pages is enabled):** https://akifnu.github.io/IELTS/

If the UI looks old, hard-refresh or open in a private window — the app clears stale service-worker caches on load.

## Flashcards (Android)

A native Android flashcard app lives in [`flashcards/`](flashcards/). See [flashcards/README.md](flashcards/README.md) for build instructions.
